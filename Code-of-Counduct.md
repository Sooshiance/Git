# Code of Conduct

This project is under the `GNU` LICENSE ***GPL3***, so everyone can contribute to it. To enhance what you do, If you wanna work on something, follow these steps:

- create new branch like this:

        git switch -c <add/commit-edit-flag>

- create new branch for removing something:

        git switch -c <perf/optimize-pull-connection-host>

and than, complete your changes, and add your code and lastly send your pull request
